# ImgurUnblocker

Replaces all imgur image links with the duckduckgo proxy.

Imgur is currently unoperational in the UK and will display "Content not viewable in your region" which plagues the internet, this fixes it by making all Imgur links be proxied with the duckduckgo proxy.

(!) It only works on embedded content and direct links; it's not possible to get images from an album.

Install: https://chromewebstore.google.com/detail/imgur-unblocker/gnfldmcodokkpcejgdlffnjakifemick
