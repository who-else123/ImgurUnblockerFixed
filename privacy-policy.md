Privacy Policy for Imgur Unblocker (Imgup)
Last updated: 11 July 2026

**1. Overview**

Imgur Unblocker (Imgup) is a Chrome extension whose single purpose is to make Imgur images viewable by redirecting matching Imgur requests through the imgup.uk image proxy.

**2. Data handled by the extension**

The extension handles limited website and browsing data solely to provide its image-proxying feature.

The extension:

- Examines image and resource URLs on webpages to determine whether they are Imgur URLs.
- Observes newly added images so that dynamically loaded Imgur images can be detected.
- Redirects matching Imgur URLs through the imgup.uk proxy.
- Stores a local counter showing how many Imgur images have been detected or proxied.

The extension does not store the contents of webpages, a list of websites visited, form entries, passwords, messages, or search history.

**3. Local storage**

The extension stores only a numerical total of detected or proxied Imgur images using Chrome's local extension storage.

This information:

- Remains on the user's device.
- Is used only to display the total inside the extension popup.
- Is not transmitted to the extension developer.
- Can be removed by clearing the extension's storage or uninstalling the extension.

**4. Information sent to imgup.uk**

When an Imgur resource is proxied, the original Imgur URL is sent to:

https://imgup.uk

This transfer is necessary to retrieve and display the requested image.

As with an ordinary HTTPS connection, imgup.uk may receive information such as:

- The user's IP address.
- Browser or user-agent information.
- The date and time of the request.
- The requested proxy URL, which contains the original Imgur URL.

The extension developer does not operate or control imgup.uk. Data handled by imgup.uk is governed by its own privacy policy:

https://imgup.uk/legal/privacy

According to that policy, imgup.uk may use infrastructure providers including IONOS and, where enabled, Cloudflare.

**5. How information is used**

Information handled by the extension is used only to:

- Detect Imgur resources.
- Redirect those resources through the image proxy.
- Display per-page and total proxy counts.
- Provide the extension's stated image-unblocking functionality.

The extension does not use data for advertising, profiling, marketing, credit decisions, or unrelated purposes.

**6. Sharing and sale of data**

The extension developer does not sell user data.

Matching Imgur URLs and normal connection information are transferred to imgup.uk only because this transfer is necessary to provide the extension's proxying feature.

The extension does not transfer user data to advertising platforms, data brokers, or other unrelated third parties.

**7. Personal information**

The extension developer does not collect names, email addresses, account details, payment information, health information, authentication credentials, personal communications, or precise GPS location.

No user account or registration is required.

**8. Security**

Requests to imgup.uk are sent using HTTPS.

The extension does not download or execute remotely hosted JavaScript or other remote executable code.

**9. Chrome Web Store Limited Use disclosure**

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

User data is used only to provide or improve the extension's single stated purpose. User data is not used for personalized advertising, sold to third parties, or used to determine creditworthiness.

**10. Changes to this policy**

This policy may be updated if the extension's functionality or data-handling practices change. The "Last updated" date will be changed whenever the policy is updated.
